# 🛡️ Go Backup Tool

[![Thai](https://img.shields.io/badge/lang-th-red.svg)](READMEth.md)
[![English](https://img.shields.io/badge/lang-en-blue.svg)](README.md)


เครื่องมือสำรองข้อมูลขนาดเล็กที่มีประสิทธิภาพสูง พัฒนาด้วยภาษา Go สำหรับการบีบอัดโฟลเดอร์ (Zip) อัตโนมัติผ่าน Command-line ช่วยให้การจัดเก็บข้อมูลของคุณเป็นระเบียบและปลอดภัย

## ✨ คุณสมบัติเด่น

- ⚡ **รวดเร็ว:** ปรับแต่งมาเพื่อความรวดเร็วในการจัดการไฟล์โดยใช้ประสิทธิภาพของภาษา Go
- 🧹 **ระบบกรองไฟล์อัจฉริยะ (Smart Filter):** ข้ามไฟล์ที่ไม่จำเป็นโดยอัตโนมัติ เช่น `.git`, `node_modules`, และไฟล์โปรแกรมต่างๆ เพื่อให้ไฟล์ Backup มีขนาดเล็กที่สุด
- 📍 **เลือกที่เก็บได้ตามใจ:** สามารถระบุโฟลเดอร์ปลายทางที่ต้องการบันทึกไฟล์ได้ (เช่น Flash Drive, External Hard Drive หรือ Desktop)
- 📅 **ตั้งชื่อไฟล์อัตโนมัติ:** ระบบตั้งชื่อไฟล์ตามชื่อโฟลเดอร์ต้นทาง พร้อมระบุวันที่และเวลาที่สำรองข้อมูลอย่างแม่นยำ
- 🔒 **ตรวจสอบความถูกต้อง:** มีการคำนวณค่า **SHA-256 Checksum** ทุกครั้งหลัง Backup เสร็จ เพื่อยืนยันว่าข้อมูลของคุณสมบูรณ์ 100%
